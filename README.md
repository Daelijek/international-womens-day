# Поздравительная страничка с 8 марта! 🌸

Красивая анимированная веб-страница для поздравления с Международным женским днём от разработчиков BeyimTech.

## ✨ Особенности

- 🎨 Красивый дизайн с градиентными фонами и анимациями
- 🌸 Анимированные падающие цветочки
- 💐 Карточки с поздравлениями с плавными переходами
- 🎲 Генератор случайных поздравлений
- 🌍 Поддержка трёх языков: казахский, русский, английский
- 📱 Адаптивный дизайн для всех устройств
- ✨ Плавные анимации и эффекты

## 🚀 Быстрый старт

### Установка зависимостей

```bash
npm install
```

### Запуск в режиме разработки

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

### Сборка для продакшена

```bash
npm run build
npm start
```

## 📁 Структура проекта

```
app/
├── components/          # Компоненты страницы
│   ├── header/         # Шапка сайта
│   ├── hero/           # Главная секция с заголовком
│   ├── greetings/      # Карточки с поздравлениями
│   ├── generator/      # Генератор поздравлений
│   ├── intro/          # Вступительный экран
│   └── language-switcher/  # Переключатель языков
├── contexts/           # React контексты
│   └── LanguageContext.tsx  # Контекст для локализации
├── lib/                # Утилиты
│   └── translations.ts # Переводы текстов
├── globals.css         # Глобальные стили и анимации
├── layout.tsx          # Основной layout
└── page.tsx            # Главная страница
```

## 🛠 Технологии

- **Next.js 16** - React фреймворк
- **TypeScript** - Типизированный JavaScript
- **Tailwind CSS** - Утилитарный CSS фреймворк
- **React Context API** - Управление состоянием локализации

## 📝 Изменение текстов

Все тексты находятся в файле `app/lib/translations.ts`. Там можно добавить новые поздравления или изменить существующие.

## 🎨 Настройка стилей

Глобальные стили и анимации находятся в `app/globals.css`. Там можно настроить цвета, анимации и другие визуальные эффекты.
